Snake

Juego de Snake creado con HTML5 usando Canvas

Tecnologias usadas: HTML5. Canvas. Javascript (ES6).

El archivo "game-dist.js" contiene toda la traduccion de las funciones usadas de ECMASCRIPT 6 a ECMASCRIPT 5 ese archivo es el que esta linkeado al html. Preprocesador: -Prepros.

Reglas:

  • Juego clasico de snake, tienes que ir comiendo todos los cuadros de comida para seguir avanzando y crecer si chocas contigo mismo pierdes, si chocas en los bordes pierdes. Cada 5 segundos aparecera comida nueva y la que estaba se borrara, si no eres rapido puedes perder!
